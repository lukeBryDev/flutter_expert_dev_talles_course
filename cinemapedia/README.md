# cinemapedia

Uses flutter v3.19.3

# Dev
1. Copiar al .env.template y renombrarlo a .env
2. Cambiar las variables de entorno (The MovieDB)
3. Cambios en la entidad, hay que ejecutar el comando
```
flutter pub run build_runner build
```
